## 스크린샷 가이드라인

### 크기
- 최대 너비: 1200px
- 파일 크기: 500KB 이하
- 포맷: PNG (UI), JPG (사진)

### 내용
- 민감한 정보 (API 키, URL 등) 모자이크 처리
- 다크 모드 또는 라이트 모드 명시
- 화면 비율 유지

### 파일명
- 소문자, 하이픈 사용
- 설명적인 이름 (예: `tts-reader-playing.png`)
- 버전 포함 시: `feature-v5.0.png`
