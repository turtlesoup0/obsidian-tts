# TTS (Text-to-Speech) 샘플 노트

> 이 노트는 Obsidian TTS 기능을 테스트하기 위한 샘플입니다

## 개요

**TTS (Text-to-Speech)**는 컴퓨터가 텍스트를 읽고 인간의 음성으로 변환하는 기술입니다.

### 주요 특징

1. **고품질 음성**: Azure Neural Voice를 사용하여 자연스러운 한국어 음성 생성
2. **볼드 강조**: **중요한 텍스트**는 음성에서도 강조되어 읽힙니다
3. **빠른 변환**: 서버 캐싱으로 재생성 불필요

## 작동 원리

TTS 시스템은 다음 단계로 작동합니다:

1. **텍스트 분석**: 입력된 텍스트를 분석하고 음소로 변환
2. **음성 합성**: AI 모델을 사용하여 자연스러운 음성 생성
3. **출력**: 생성된 오디오 파일 재생

## 볼드 강조 테스트

이 문장에서 **이 부분만** 강조되어 읽힙니다.

일반 텍스트와 **강조 텍스트**를 섞어서 사용할 수 있습니다.

**전체 문장을 강조**할 수도 있습니다.

## 긴 문장 테스트

인공지능 기술의 발전으로 텍스트 음성 변환 기술은 점점 더 자연스러워지고 있습니다.
과거에는 로봇 같은 목소리였지만, 현대의 신경망 기반 TTS는 사람의 억양과 감정까지 표현할 수 있습니다.

## 기술 용어 테스트

- **API (Application Programming Interface)**: 애플리케이션 프로그래밍 인터페이스
- **JSON (JavaScript Object Notation)**: 자바스크립트 객체 표기법
- **HTTP (Hypertext Transfer Protocol)**: 하이퍼텍스트 전송 프로토콜
- **URL (Uniform Resource Locator)**: 통합 자원 위치 지정자
